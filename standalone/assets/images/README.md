# Images Directory

Place your logo files here.

## Supported Formats
- PNG (recommended for logos with transparency)
- SVG (recommended for scalable logos)
- JPG/JPEG

## Usage

To use a logo in the application:

1. Place your logo file in this directory (e.g., `logo.png` or `logo.svg`)
2. Uncomment the logo line in `index.html`:
   ```html
   <img src="assets/images/logo.png" alt="Logo" class="header-logo">
   ```
3. Update the filename to match your logo file

## Recommended Logo Specifications
- **Height**: 60px (will be scaled automatically)
- **Format**: PNG with transparency or SVG
- **Aspect Ratio**: Flexible, but square or wide formats work best
